# E-Commerce API

## About

A RESTful API built with Node.js, Express, and MongoDB for managing products and shopping carts.

## Features 

- Product CRUD
- Cart API
- Express Router
- Controllers
- Error Middleware
- Postman Collection

## Installation

npm install

npm run dev

## API

GET /products

POST /products

PUT /products/:id

DELETE /products/:id

POST /carts

GET /carts/:id

POST /carts/:id/items
